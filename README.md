# RSX Crosshair
A lightweight RedM crosshair selector resource with a clean NUI menu and 12 selectable crosshair styles. Players can open the menu with a configurable command, choose their preferred crosshair, and display it while aiming.

## Features

- 12 built-in crosshair images
- Configurable menu command
- Configurable default crosshair
- Optional custom crosshair toggle
- Simple NUI selector menu
- Lightweight client-side resource
- RedM no-red-dot aiming support

## Installation

1. Download or clone this resource.
2. Place the `rsx_crosshair` folder inside your server `resources` folder.
3. Add this to your `server.cfg`:

```cfg
ensure rsx_crosshair
```

4. Restart your server or run:

```cfg
refresh
ensure rsx_crosshair
```
